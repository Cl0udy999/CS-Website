Tip Calculator

Harvard.EDX.org/TipCalculator

As a prerequisite for my CS class, I learned Python through Harvard EDX. This is an assignment that I coded for that course and it's able to calculate tip amounts. First, the program prompts the user to input an amount and the percent they want to tip. Then, it calculates that amount by multiplying the dollar amount and the percentage. This program is able to calculate any amount with any percentage and is useful if you’re not sure how much to tip.

Critter Clean-Up Game

snap.berkeley.edu/anika.crittercleanup

In a block code program, me and my partner created a game that teaches young children how to recycle and care for the species that are affected by littering. Our main character, Barry the Bear, shares the experiances of wild animals and teaches kids which types of items they see everyday can be recycled. Through a drag-and-drop interface, kids can participate in the activity of recycling and earn points for their efforts!
